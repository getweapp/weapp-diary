# 微信小程序－心情日记

### 说明：

实现写日记功能（注意点击保存时要先让内容框失去焦点），特色：
- 文本框自动变动高度
- 小程序缓存的使用

### 数据接口:

使用本地数据

### 目录结构：

- pages — 存放项目页面文件
- utils — 存放日期处理文件

### 开发环境：

微信web开发者工具 v0.11.112301
